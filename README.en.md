# 硬件查看器

#### Description
本工具专为Linux系统设计，提供硬件及驱动信息的可视化查看功能。
支持系统概览、CPU、内存、磁盘、网卡、PCI设备、USB设备
和显卡等硬件信息的展示，并提供明暗主题切换功能。
